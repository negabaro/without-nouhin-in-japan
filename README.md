# without-nouhin-in-japan
**納品のない受託開発**をしている日本の会社リスト

リストはこちらの[納品のない受託開発会社リスト](/companies.md)リンクをご確認ください。

## Purpose
システム開発を発注したい企業、特にスタートアップや新規事業開発において、先の読めない市場で素早く仮説検証をして仕様変更やビジネスのピボットをする必要があります。
そのような企業においてシステム開発を外注する場合、納品型の受託開発はあまりフィットせず、成果物よりサービスの成長に対してコミットする**納品のない受託開発**がよりマッチします。

しかしながら意外なことにあまり知られていません。そこで本リストを作りPRでリストを拡張することで、納品をしない受託開発をする会社の認知度を上げて行きたいと考えています。
また納品型の受託開発にほとほと懲りてしまったソフトウェアエンジニアに転職先の候補として知ってもらいたい。

## Contributing
Pull Request大歓迎です。以下の条件を満たしていれば、積極的にマージします。新しい会社は[納品のない受託開発会社リスト](/companies.md)のテーブルに追記してください。

* システム開発を行う企業である
* 準委任契約またはそれに任じる契約で顧客のサービスの成功に寄与する開発を行う(ただし請負契約を絶対しない必要無し)
* 日本に拠点がある
